Для работы программы необходимы
следующие python библеотеки:

1. eel  -------------------------------- (pip install eel)
2. tkinter ------------------------------ (pip install tk)
3. PIL ------------------------------ (pip install Pillow)
4. pydub ----------------------------- (pip install pydub)

----------------------------------------------------------

Для корректной работы с аудиофайлами необходимо установить 
ffmpeg (https://github.com/BtbN/FFmpeg-Builds/releases)
и добавить ее в глобальную переменную
PATH (https://www.java.com/ru/download/help/path.html)
